# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If a vunerability was introduced in an a recent release, you can start talking about it on [that release's announcement](https://github.com/ArielMAJ/test.me/discussions/categories/announcements).
You can also report vunerabilities through [issues](https://github.com/ArielMAJ/test.me/issues) or starting new [discussions](https://github.com/ArielMAJ/test.me/discussions).
After opening an issue, feel free to start working on it and starting a Pull Request that solves that problem.
